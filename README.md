# SC-55
Attempt to emulate SC-55mkII

## Status

3 ICs are needed to be emulated:
- Roland PCM chip
- Hitachi H8/532 MCU
- Mitsubishi M37450M2 MCU

MCUs will be emulated using info from their datasheets. PCM chip info will be derived from analyzing decapped chip.
